# 하노이의 탑
- 쌓아놓은 원반을 최소 횟수로 옮기는 알고리즘
```
원판 N개를  A -> C 로 옮기려면
먼저 위의 N-1개를 A -> B 로 옮기고,
가장 큰 원판을 A -> C로 옮기고
N-1개를 B -> C로 옮긴다
```
# 8퀸 문제
```
퀸은 가로, 세로, 대각선 어느 방향으로도 공격할 수 있음.
```

- 분기 작업 : 배치 조합을 열거하는 방법
- 분할 정복법 : 큰 문제를 작은 문제로 분할, 작은 문제 풀이법을 결합하여 전체 풀이법을 얻는 방법
- 한정 작업 : 필요하지 않은 분기를 없애서 불필요한 조합을 열거하지 않는 방법
- 분기 한정법 : 분기 작업과 한정 작업을 조합하여 문제를 풀이하는 방법, 경로는 최적해가 될 수 없어 포기
- **백트래킹** : 이 경로는 해가 될 수 없기에 즉시 포기
* 분기 한정법 != 백트래킹

```
가능한 모든 경우를 탐색하는 완전 탐색,
한정 작업으로 탐색 횟수 줄이기. ex) 1행 1열 퀸일 경우, 2행 1열 퀸은 불가. 그 이후 작업 경우의 수 버리기
```
