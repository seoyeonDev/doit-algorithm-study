# 알고리즘 기초
## 알고리즘이란?
- **정의** : 어떠한 문제를 해결하기 위해 정해놓은 일련의 절차
- **올바른 알고리즘** : 어떠한 경우에도(예외 없이) 기대한 결과가 정확히 나오는 것

- 기본 구조
    - **순차 구조** : 위에서 아래로 순서대로 처리되는 구조
    - **선택 구조** : 조건에 따라 실행 흐름이 바뀌는 것 

<br/>

## 조건문과 분기

### 복합문의 구조
`if`, `while`처럼 키워드로 시작해 `:`으로 끝나는 **Header**와, 그 아래 들여쓰기 된 실행부인 **Suite**로 구성

#### if문
```
if 조건식1:
    # 조건식1이 참일 때 실행될 스위트(Suite)
elif 조건식2:
    # 조건식1이 거짓이고, 조건식2가 참일 때 실행될 스위트
else:
    # 위의 모든 조건이 거짓일 때 실행될 스위트
```

#### while문
```
while 조건식:
    # 조건식이 참인 동안 반복해서 실행될 스위트
else:
    # 조건식이 거짓이 되어 반복이 정상 종료된 후 실행될 스위트 (선택 사항)
```

<br/>

## 반복하는 알고리즘

- **반복 구조(repetition structure)** : 어떤 조건이 성립하는 동안 반복해서 특정 명령을 **되풀이** 하는 것
- 반복의 도구
    - `while` : 조건이 참인 동안 무한히 반복 가능
    - `for` : 정해진 범위나 객체를 순회할 때 사용 (변수가 하나만 있을 때는 while문보다 for문을 사용하는 것이 좋다)
    - `range()` : 0부터 n-1까지 숫자를 생성하는 이터러블 객체

<br/>

### range() 함수
`range()` 함수는 이터러블 객체를 생성한다. <br/>
이터러블 객체는 반복할 수 있는 객체를 말한다. <br/>
대표적인 자료형으로 `list, str, tuple`이 있다. <br/>

- `range(n)` : `0` 이상 `n` 미만인 수를 차례로 나열하는 수열
- `range(a, b)` : `a` 이상 `b` 미만인 수를 차례로 나열하는 수열
- `range(a, b, step)` : `a` 이상 `b` 미만인 수를 `step` 간격으로 나열하는 수열

<br/><br/>

# 참고 정리

## 부동 소수점 방식
부동 소수점(floating point)은 컴퓨터에서 실수를 근삿값으로 표현할 때 사용함.<br/>
부동 소수점 방식 : **가수 부분**과 **지수 부분**으로 나누어 표현하는 것을 말한다.

### 0.1 + 0.2 는 0.3이 아니다
컴퓨터는 모든 숫자를 이진수로 처리한다.<br/>
그러나 10진수 0.1은 이진수로 바꾸면 무한 소수가 되어 끝없이 반복이 된다.<br/>
따라서 `0.1` 과 `0.2`를 더하게 되면 `0.30000000000000004`가 된다.<br/>

```
print(0.1+0.2==0.3)
// false
// 0.1+0.2=0.30000000000000004
```

### 비교를 하기 위해서는?
- ① **math.isclose()** 사용 (비교할 때) <br/>
두 숫자가 "거의 같은지" 확인하는 함수를 사용한다.
```
import math
print(math.isclose(0.1 + 0.2, 0.3))  # True
```

- ② **decimal** 모듈 사용 (정확한 계산이 필요할 때) <br/>
돈 계산처럼 정확히 계산을 진행해야 할 때는 float 대신 decimal.Decimal을 사용해야 한다.

```
from decimal import Decimal
print(Decimal('0.1') + Decimal('0.2') == Decimal('0.3'))  # True
```

## f-string(포맷 스트링)

```
print(f"1부터 {n}까지 정수의 합은 {sum}입니다.")
```
### f-string(포맷 스트링) 사용 방법

1. 따옴표 앞머리에 `f` 또는 `F`를 붙인다.
2. 변수나 식을 넣고 싶은 곳에 **중괄호** `{ }`를 사용한다.
