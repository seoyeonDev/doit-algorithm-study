# 01-1 알고리즘이란?
알고리즘: 어떠한 문제를 해결하기 위해 정해놓은 일련의 절차
= 어떠한 경우에도 실행 결과가 똑같이 나오는 것

structured programming (구조적 프로그래밍) 
: 입력과 출력으로 이루어진 구성 요소를 계층으로 배치하여 프로그램을 구성
: 세 종류의 제어 흐름을 사용 -- 순차, 선택, 반복

- sequential structure (순차 구조) : 한 문장씩 순서대로 처리
- select structure (선택 구조) : 조건식 결과에 따라 프로그램의 실행 흐름 변경
- repetition structure (반복 구조) : 어떤 조건이 성립하는 동안 반복해서 처리 (loop)

decision tree (결정 트리) : 조합을 나열한 모습



## 세 정수의 최댓값 구하기

```python
# 세 정수의 최댓값 구하기
def max3(a, b, c):
    maximum = a
    if b > maximum: maximum = b
    if c > maximum: maximum = c
    return maximum  # 최댓값 반환

print(f'max3(3, 2, 1) = {max3(3, 2, 1)}')   # max3(3, 2, 1) = 3
```

```python
# 세 정수를 입력받아 중앙값 구하기 1
def med3(a, b, c):
    if a >= b:
        if b >= c:
            return b
        elif a <= c:
            return a
        else:
            return c
    elif a > c:
        return a
    elif b > c:
        return c
    else:
        return b

a = int(input('a: '))   # 문자열을 정수로 형 변환
b = int(input('b: '))
c = int(input('c: '))
print(f'중앙값: {med3(a, b, c)}')
```

## 조건문과 분기
operator (산술 연산자) : + 나 - 등의 기호
operand (피연산자) : 연산 대상

연산자는 피연산자의 개수에 따라 3가지로 분류됨
1. unary operator (단항 연산자) : 피연산자 1개 (ex. -a)
2. binary operator (이항 연산자) : 피연산자 2개 (ex. a < b)
3. ternary operator (삼항 연산자) : 피연산자 3개 (ex. a if b else c)
    - conditional operator (조건 연산자) : 파이썬의 유일한 삼항 연산자 (if~else)

# 01-2 반복하는 알고리즘

## 1부터 n까지 정수의 합 구하기
### while 문 반복
판단 반복 구조 : while 문은 실행하기 전 반복을 계속할 것인지 판단

```python
n = int(input('n: '))

sum = 0
i = 1   # 카운터용 변수 : 반복 제어용

while i <= n:   # 판단 반복 구조 : 반복을 계속할 것인지 판단
    sum += i
    i += 1

print(f'1부터 {n}까지 정수의 합: {sum}')
```

### for 문 반복
변수가 하나만 있을 때는 while 문 보다 for 문이 좋다.
```python
n = int(input('n: '))

sum = 0

for i in range(1, n + 1):
    sum += i

print(f'1부터 {n}까지 정수의 합: {sum}')
```

## 연속하는 정수의 합을 구하기 위해 값 정렬하기
### range() 함수로 이터러블 객체 생성
이터러블 객체 : 반복할 수 있는 객체
대표적인 이터러블 자료형 : list, str, tuple

- range(n) : 0 이상 n 미만인 수를 차례로 나열
- range(a, b) : a 이상 b 미만인 수를 차례로 나열
- range(a, b, step) : a 이상 b 미만인 수를 step 간격으로 나열

```python
# a부터 b까지 정수의 합 구하기 1
a = int(input('a: '))
b = int(input('b: '))

# a와 b를 오름차순으로 정렬
if a > b:
    a, b = b, a    # a와 b의 값을 교환 (단일 대입문)  

sum = 0
for i in range(a, b):
    sum += i

sum += b

print(f'{a}부터 {b}까지 정수의 합: {sum}')
```

```python
# 두 자리 양수만 입력받기
while True:
    n = int(input('n: '))

    # 1) 비교 연산자를 연속으로 사용
    if n >= 10 and n <= 99:     
        break

    # 2) 드모르간의 법칙
    # : 각 조건을 부정한 뒤, 논리곱->논리합 논리합->논리곱 으로 바꾼 뒤 다시 전체를 부정하면 원래 조건과 같아
    if not(n < 10 or n > 99):     
        break
```


## break & continue
```python
# 가로, 세로 길이가 정수이고 넓이가 area인 직사각형에서 변의 길이 나열
area = int(input('직사각형 넓이: '))

for i in range(1, area + 1):
    if i * i > area: break
    if area % i: continue
    print(f'{i} x {area // i}')
```
