# 스택 (Stack)

## 개요

**스택**은 데이터를 임시로 저장할 때 사용하는 자료구조이다. <br/>
데이터의 입력과 출력은 **후입선출(LIFO, Last In First Out)** 방식을 따름<br/>

---

## 기본 연산

| 연산 | 설명 |
|------|------|
| `push` | 스택에 데이터를 넣기 |
| `pop` | 스택에서 데이터를 꺼내기 |

---

## 주요 메서드

### `push()`
스택에 데이터를 추가
- 스택이 가득 찬 경우 `FixedStack Full` 예외를 발생
- 데이터 추가 후 스택 포인터 `ptr`을 1 증가

### `pop()`
스택의 꼭대기에서 데이터를 꺼내어 반환
- 스택이 비어 있는 경우 예외를 발생

### `peek()`
스택의 꼭대기 데이터를 꺼내지 않고 확인
- 스택이 비어 있는 경우 예외를 발생

### `clear()`
스택의 모든 데이터를 삭제
- 스택 포인터 `ptr`의 값을 `0`으로 초기화

### `find()`
`value`와 동일한 데이터가 스택에 존재하는지 확인하고, 존재한다면 배열 내 위치를 반환
- 데이터가 없을 경우 `-1`을 반환

### `count()`
스택에 현재 쌓여 있는 데이터의 개수를 반환

### `__contains__()`
스택에 특정 데이터가 존재하는지 판단
- 존재하면 `True`, 없으면 `False`를 반환

### `__len__()`
클래스 인스턴스를 `len()` 함수에 전달할 수 있도록 스택의 길이를 반환
