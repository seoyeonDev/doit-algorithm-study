# 자료구조와 배열

학생들의 점수를 총 합하고 평균을 구해야 할 때 묶음 단위로 값을 저장하는 **배열**이라는 자료구조를 다뤄야 한다.  
파이썬에서는 리스트와 튜플이 이 역할을 대신한다.

- **배열과 객체**: 배열에는 객체가 저장되며, 배열에 저장된 객체 하나하나를 **원소(element)** 라고 한다.
- **인덱스(Index)**: 각 원소는 0, 1, 2 순으로 인덱스를 부여받는다.

**배열이 필요한 경우:**
- 학생 수를 변경하는 경우 (length가 변하는 경우)
- 특정 학생의 시험 점수를 확인하거나 변경하는 경우 (index가 필요한 경우)
- 최저점과 최고점을 구하거나 정렬이 필요한 경우

# 리스트와 튜플

리스트와 튜플은 데이터를 담는 컨테이너이다.  
가장 큰 차이점은 원소를 변경할 수 있는지(**Mutable**) 없는지(**Immutable**)이다.
<br/><br/>

## 1. 리스트 (List)

리스트는 원소를 변경할 수 있는 **뮤터블(mutable)** 객체이다.

``` 
# 기본 생성
list01 = []
list02 = [1, 2, 3]
list03 = ['A', 'B', 'C']

# list() 내장 함수 사용
list04 = list()          # [] 빈 리스트
list05 = list('ABC')     # ['A', 'B', 'C'] 문자열의 각 문자로 원소 생성
list06 = list([1, 2, 3]) # [1, 2, 3] 리스트로부터 생성
list07 = list((1, 2, 3)) # [1, 2, 3] 튜플로부터 생성
list08 = list({1, 2, 3}) # [1, 2, 3] 집합으로부터 생성

# range() 활용
list09 = list(range(7))        # [0, 1, 2, 3, 4, 5, 6]
list10 = list(range(3, 8))     # [3, 4, 5, 6, 7]
list11 = list(range(3, 13, 2)) # [3, 5, 7, 9, 11]

# 원솟값을 정하지 않고 공간만 할당
list12 = [None] * 5
```
<br/><br/>

## 2. 튜플 (Tuple)

원소에 순서를 매겨 결합한 것으로, 원소를 변경할 수 없는 **이뮤터블(immutable)** 객체이다.

```
# 기본 생성
tuple01 = ()
tuple02 = 1,        # 원소가 한 개인 경우 쉼표(,) 필수! 없으면 단순 변수로 인식
tuple03 = (1,)
tuple04 = 1, 2, 3
tuple05 = 1, 2, 3,
tuple06 = (1, 2, 3)
tuple07 = (1, 2, 3, )
tuple08 = 'A', 'B', 'C',

# 내장 함수 및 range 활용
tuple09 = tuple()
tuple10 = tuple('ABC')
tuple11 = tuple([1, 2, 3])
tuple13 = tuple(range(7))
```

# 데이터 접근하기

## 1. 언팩 (Unpack)

리스트나 튜플의 원솟값들을 풀어 여러 변수에 대입하는 것이다.

```
x = [1, 2, 3]
a, b, c = x  # a=1, b=2, c=3
```
<br/><br/>

## 2. 인덱스로 접근

```
x = [11, 22, 33, 44, 55, 66, 77]
x[-4] = 3.14  # 뒤에서 4번째(44)가 3.14로 변경됨
# x[7] = 3.14 # IndexError: 존재하지 않는 인덱스이므로 오류 발생
```
<br/><br/>

## 3. 슬라이스 (Slice)

리스트나 튜플의 일부를 꺼내 새로운 객체를 만든다. `s[i:j:k]`는 i부터 j-1까지 k씩 건너뛰며 나열한다.

```
s = [10, 20, 30, 40, 50, 60, 70]

s[0:6]    # [10, 20, 30, 40, 50, 60]
s[0:7:2]  # [10, 30, 50, 70]
s[-4:-2]  # [40, 50]
s[-4:2]   # [] (시작점이 끝점보다 뒤에 있으므로 빈 리스트 반환)

# 슬라이스 주요 패턴
s[:]      # 전체 출력
s[:n]     # 맨 앞부터 n개까지
s[i:]     # i부터 끝까지
s[-n:]    # 뒤에서 n개부터 끝까지
s[::k]    # 처음부터 k개씩 건너뛰며
s[::-1]   # 역순 출력
```
<br/><br/>

## 4. 배열 원소를 역순으로 정렬

역순 정렬에는 두 가지 방법이 있다.

```
x.reverse()            # 원본을 직접 수정
y = list(reversed(x))  # 원본 그대로, 새로운 리스트 생성
```

> **reversed()가 필요한 이유?**  
> `reverse()`는 원본을 영구적으로 바꾸기 때문에, 원본을 보존하면서 역순 데이터가 필요할 때는 `reversed()`를 사용한다.

# 파이썬의 핵심 개념

## 문(Statement)과 식(Expression)

- **식 (Expression)**: 값을 산출하는 코드. (예: `x + 17`)
- **문 (Statement)**: 실행 가능한 최소 단위. (예: `x = 17`)

> 파이썬에서 `=`는 연산자가 아니라 대입을 수행하는 '문'의 기호이다.

<br/><br/>

## 등가성(==)과 동일성(is)

- **등가성 (`==`)**: 두 객체의 **값**이 같은지 비교한다.
- **동일성 (`is`)**: 두 객체의 **식별 번호(메모리 주소)** 가 같은지 비교한다.

```
list1 = [1, 2, 3]
list2 = [1, 2, 3]
list1 == list2  # True  (값이 같음)
list1 is list2  # False (서로 다른 메모리 위치에 존재)

list3 = list1
list1 is list3  # True  (같은 메모리 주소를 가리킴)
```

# 활용: 최댓값 구하기

`typing` 모듈의 `Any`와 `Sequence`를 사용하여 범용적인 함수를 만들 수 있다.

```
from typing import Any, Sequence

def max_of(a: Sequence) -> Any:
    """시퀀스 a의 최댓값을 구하여 반환"""
    maximum = a[0]
    for i in range(1, len(a)):
        if a[i] > maximum:
            maximum = a[i]
    return maximum

if __name__ == "__main__":
    num = int(input("원소 수 입력: "))
    x = [None] * num
    for i in range(num):
        x[i] = int(input(f"x[{i}]값을 입력하세요: "))
    print(f"최댓값은 {max_of(x)} 입니다.")
```
