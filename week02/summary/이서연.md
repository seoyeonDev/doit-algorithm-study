


## 배열 
- 묶음 단위로 값을 저장
### 원소 
- 배열에 저장된 객체 하나하나
### 인덱스 
- 원소에 부여된 순서 

## 데이터 컨테이너 
### 리스트 
- 원소를 변경할 수 있는 뮤터블 list 형 객체, 
- []안에 원소를 쉼표로 구분하여 표기 
- 파이썬에서 내장함수인 list() 사용 
- 범위로 구성된 리스트 range(). list() 조합 
### 튜플 
- 원소를 변경할 수 없는 이뮤터블 자료형, () 로 둘러싸는 방식에 쉼표로 구분 
- 내장함수인 tuple() 사용 

### 언팩
- 좌변에 여러개의 변수, 우변에 리스트나 튜플 놓으면 우변의 원소를 좌변의 변수에 한번에 대입 가능.
```
x = [1,2,3]
a,b,c = x

a,b,c 
=> (1,2,3)
```
- 리스트나 튜플의 원솟값들을 풀어 여러 변수에 대입하는 것

### 인덱스
- 양수 인덱스 : 왼쪽 끝부터 0부터 1씩 증가 
- <U>음수 인덱스 : 0 - 전체 원소 수부터 1씩 증가</U>

<U>값을 대입할 떄, 값 자체가 복사되는 것이 아니라 참조하는 객체의 식별 번호가 변경됨</U>

### 슬라이스
- 리스트, 튜플의 원소 일부를 연속해서/일정한 간격으로 새로운 리스트,튜플을 만드는 것 
- <U>자바의 subString,subList 와 유사</U>
```python
s[i:j] # i부터 j-1까지 나열
s[i:j:k]
```


<br></br>

- 파이썬은 대입 기호 =를 연산식에서 사용하는 +,* 등과 다르게 취급


### 순차 반복 객체
- 내부의 요소들을 한 번에 하나씩 차례대로 뽑기
